# DayBook - 个人日报博客

一个简洁高效的个人日报博客网站，支持每日记录、AI 周报生成、笔记管理，采用温暖的「晨间咖啡馆」主题设计。

## 功能特性

- **日报管理** - 记录每天的工作日报，支持标题、内容、心情、标签
- **AI 周报生成** - 根据本周日报自动生成周报摘要
- **笔记系统** - 分类管理笔记，支持标签筛选
- **贡献图** - 可视化展示每日记录情况

## 技术栈

- **框架**: Next.js 16 + React 19
- **UI**: shadcn/ui + Tailwind CSS
- **数据库**: Supabase (PostgreSQL)
- **AI**: Coze Coding SDK

## 快速开始

```bash
pnpm install
pnpm dev
```

访问 http://localhost:5000

## 设计主题

「晨间咖啡馆」- 温暖而专业的设计风格

- 植物绿植 (#5B8C5A) - 主色调
- 焦糖咖啡 (#D4A574) - 强调色
- 米白纸张 (#FAF8F5) - 背景色
